# Distribuição Gama e Análise NHANES

Projeto desenvolvido no âmbito da unidade curricular **Laboratórios de Estatística II**.

## Descrição

Este projeto divide-se em duas partes principais:

- **Parte I – Distribuição Gama:** estudo teórico e visual da distribuição Gama, incluindo parâmetros, medidas teóricas, função densidade (PDF), função acumulada (CDF), Lei dos Grandes Números e Teorema do Limite Central.
- **Parte II – Análise NHANES:** análise exploratória de dados da base **NHANES**, com medidas descritivas, gráficos e intervalo de confiança para a média do BMI.

Além disso, foi desenvolvida uma aplicação interativa em **Streamlit** para explorar os resultados de forma dinâmica.

## Ficheiros do repositório

- `app_streamlit.py` — aplicação Streamlit
- `nhanes_limpo.csv` — base de dados usada na análise NHANES
- `Projeto_Grupo.R` — script em R usado para limpeza/preparação dos dados
- `relatorio.tex` — versão LaTeX do relatório
